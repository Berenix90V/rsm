br_formfield -> basic registry formfield (gestisce il form dell'anagrafica)
br_tablearray -> basic registry tablearray (gestisce la tabella dell'anagrafica)
inreq_formfields -> richieste in entrata formfields
inreq_tablearray -> richieste in entrata tabelle
is_formfields -> In stock (in magazzino) per formfields
is_tablearray -> In stock per tabelle
outreq_formfields, outreq_tablearray -> richieste in uscita
pfr_formfields, pfr_tablearray -> relazioni prodotti,fornitori, anagrafica
req_formfields, req_tablearray -> richieste
totreq_tablearray ->  totale richieste
